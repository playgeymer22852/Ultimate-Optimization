## 🪐 О проекте / About Project
**Ultimate Fabric Optimization** — это продвинутый технический пак, созданный с одной целью: радикально повысить комфорт от игрового процесса. Сборка обеспечивает колоссальный буст FPS, полностью вырезает микрофризы при генерации мира и минимизирует потребление оперативной памяти движком Minecraft. 

> ⚠️ **ВАЖНОЕ ПРИМЕЧАНИЕ:** Этот репозиторий GitHub является исключительно **официальным центром поддержки и баг-трекером**. Сами файлы игры, обновления и загрузки публикуются строго на платформе **Modrinth**.

## 💻 Технические параметры / System Requirements

| Параметр | Минимальные | Рекомендуемые | Для тяжелых шейдеров |
| :--- | :--- | :--- | :--- |
| **Выделенная ОЗУ** | 4 ГБ | 8 ГБ | 12 ГБ + |
| **Версия Java** | Java 25 (jdk-25) | Java 25 (Adoptium) | Java 25+ |
| **Графический API** | OpenGL | OpenGL | OpenGL |

*Физические ограничения игрового движка майнкрафта не позволяют разгонять FPS бесконечно. Сборка работает практичнее — она железно стабилизирует время кадра и полностью убирает заикания при прогрузке чанков.*

***

## 📊 Состав оптимизации / Core Mod Features

Никакого лишнего контента и глобальных модов — только чистая ванилла, работающая на пределе возможностей вашего железа:

*   ⚡ **Рендеринг и FPS:** `Sodium`, `Sodium Extra`, `Reese's Sodium Options`, `ImmediatelyFast`, `BadOptimizations`.
*   🎯 **Ресурсосбережение:** `Entity Culling`, `More Culling` (отсечение невидимых полигонов и мобов за стенами).
*   🧠 **Движок и Логика:** `Lithium` (оптимизация физики), `ModernFix` (исправление утечек памяти), `FerriteCore` (сжатие RAM).
*   🗺 **Мир и Сеть:** `C2ME` (многопоточная прогрузка чанков), `Chunky`, `Dynamic FPS`, `Fast Server Pings`.
*   🔊 **3D Звук:** `Sound Physics Remastered`, `AmbientSounds`, `Presence Footsteps` (полное аудио-погружение).

## 🪲 Как сообщить о баге / How to Report a Bug

Если у вас вылетает игра, резко упала производительность или вы нашли ошибку в конфигурации, пожалуйста, сообщите об этом нам:

1. Перейдите в верхнем меню этого репозитория во вкладку **Issues (Проблемы)**.
2. Нажмите зелёную кнопку **New issue** (Новая проблема).
3. Выберите нужную интерактивную веб-форму (*Отчёт об ошибке*, *Запрос нового мода* или *Изменение настроек*).
4. Заполните обязательные поля (включая характеристики вашего ПК и лаунчер) и нажмите **Submit new issue**.
